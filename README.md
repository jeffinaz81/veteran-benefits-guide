# Veteran Benefits Guide

A free, simple web tool that helps U.S. military veterans estimate VA disability compensation and other military benefits.

## 🔍 What this site does

This website provides an easy-to-use calculator for veterans to estimate monthly VA disability pay based on disability rating and dependents.

It is designed to be:
- Fast
- Mobile-friendly
- Easy to use
- Free for all veterans

## Features

- VA Disability Compensation Calculator
- Simple input-based interface
- Instant estimated monthly results
- Educational information about VA disability ratings

## Purpose

This project was created to help veterans better understand and estimate their potential VA disability compensation in a simple, accessible way.

## Disclaimer

This tool provides estimates only and is not affiliated with the U.S. Department of Veterans Affairs (VA). Actual compensation may vary.

## Future Plans

This site will expand to include additional veteran benefit calculators such as:
- Military retirement pay estimator
- GI Bill housing allowance calculator
- VA rating breakdown tools
